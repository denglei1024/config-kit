# Config Kit 🛠️

**一套开箱即用的开发环境配置集合**  
包含 Git、Vim、SDK 工具等常用配置。

---

## 目录结构

```
config-kit/
├── git/
│ ├── .gitconfig # Git 全局配置（用户名、别名等）
│ └── .gitignore_global # 全局 Git 忽略规则
├── idea/
│ └── .ideavimrc # IdeaVim 基础配置（插件、快捷键等）
├── sdk/
│ └── global.json # SDK/CLI 工具全局配置（如 .NET Core）
└── README.md
```
